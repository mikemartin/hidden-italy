{{# @name Accordion @desc A single collapsible card. Insert multiple sets in a row to get a stack of independent accordions — the article's prose wrapper (`stack-8`) supplies the vertical gap between them, and each card owns its own Alpine `open` state so toggles never interact across cards. Collapsed by default. The closed state shows the title with a chevron; the open state reveals the `body` Bard field rendered as HTML inside a non-prose surface so its own typography rules win over the surrounding `.prose` styles. The set's `id` is used as both the panel's DOM `id` and the aria-controls target so each card is independently addressable for assistive tech and deep links. @set page.article.accordion #}}
