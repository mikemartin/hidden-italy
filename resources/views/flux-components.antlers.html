{{# Auth pages #}}

Component Used in Purpose
flux:input login, register, forgot-password, reset-password, confirm-password Text, email and password fields with built-in label and validation messaging.
flux:checkbox login "Remember me" checkbox.
flux:link login, register, forgot-password Inter-page navigation (Sign up / Log in / Forgot password) with wire:navigate.
flux:text verify-email Body copy and the verification-link-sent success notice.

{{# Account / settings pages #}}
{{ partial:typography/h2 content="Account & settings pages" class="!mb-0" }} {{ partial:typography/p content="resources/views/pages/settings/" class="!mb-0" }}
Component Used in Purpose
flux:input profile, password, postal-address, emergency-contact All settings form fields.
flux:heading every settings form sr-only form titles for assistive tech.
flux:select + flux:select.option postal-address Country and state dropdowns.
flux:navlist + flux:navlist.item settings/layout Sidebar navigation between settings sub-pages.
flux:separator settings/layout Mobile divider between nav and content.
flux:link + flux:text profile "Resend verification" link and email-unverified notice.
{{# Tabs — preview for the Experience section on the tours page #}}
{{ partial:typography/h2 content="Tabs" class="!mb-0" }} {{ partial:typography/p content="Preview of flux:tabs for the Experience section on the tours page (replaces the custom Alpine tab UI in page_builder/_experience.antlers.html). Shown below in light and dark surface modes." class="!mb-0 max-w-3xl" }}
Component Purpose
flux:tab.group Wrapper that ties tab triggers and panels together.
flux:tabs Tab list. Accepts default="…" for the initial active tab.
flux:tab Individual trigger. icon resolves to Heroicons by default, or to any Lucide icon imported via php artisan flux:icon.
flux:tab.panel Content for one tab; name matches the trigger.

{{# Accordion — preview for the FAQs page-builder block #}}
{{ partial:typography/h2 content="Accordion" class="!mb-0" }} {{ partial:typography/p content="Preview of flux:accordion for the FAQ page-builder block (page_builder/_faqs.antlers.html). Shown below in light and dark surface modes." class="!mb-0 max-w-3xl" }}
Component Purpose
flux:accordion Wrapper. Optional transition, exclusive (one open at a time), variant="reverse".
flux:accordion.item Single section. Use heading="…" shorthand or nested flux:accordion.heading + flux:accordion.content. expanded opens by default.

{{# Modal — preview for the booking enquiry auth gate #}}
{{ partial:typography/h2 content="Modal" class="!mb-0" }} {{ partial:typography/p content="Preview of flux:modal for the booking enquiry auth gate (components/_request_booking_modal.antlers.html)." class="!mb-0 max-w-3xl" }}
Component Purpose
flux:modal Modal body. name (required, unique per page), variant (default / flyout / bare), position, :dismissible="false", wire:model, @close / @cancel.
flux:modal.trigger Wraps the element that opens the modal. name matches the modal; optional shortcut.
flux:modal.close Wraps any element that should close the modal when clicked.
